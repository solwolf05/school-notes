### Electron Orbitals

Each energy level (principle quantum number n) contains up to n sublevels.  
Sublevels are: s, p, d, f.

| Energy Level | Sublevels  |
| ------------ | ---------- |
| 1            | s          |
| 2            | s, p       |
| 3            | s, p, d    |
| 4            | s, p, d, f |

Each sublevel contains a set number of orbitals.  
Each orbital holds up to 2 electrons.  
This is because they have opposite spins. No two electrons can be in the same place at the same time with the same spin (Pauli Exclusion Principle).

| Sublevel | Orbitals | Max Electrons |
| -------- | -------- | ------------- |
| s        | 1        | 2             |
| p        | 3        | 6             |
| d        | 5        | 10            |
| f        | 7        | 14            |

Electrons fill lower-energy orbitals first (Aufbau Principle).
